| Principio | Evidencia | Aplicación JarvisAI |
|-----------|-----------|---------------------|
| **3 tiers dominan** | [62% usan tiered pricing](https://www.nocodefinder.com/blog-posts/ai-agent-pricing) | ✅ Starter/Pro/Enterprise |

| **4+ tiers = confusión** | 31% peor conversión con 4+ opciones | ✅ Máximo 3 tiers |
| **Charm pricing ($9 ending)** | Aumenta percepción de valor | ✅ $79, $199, $799 |
| **Garantía visible** | Reduce fricción de compra 40%+ | ✅ "15 dias gratis o reembolso" |

## 🎯 Estrategia de Precios JarvisAI

### Posicionamiento vs Competencia

**Ventaja diferencial de JarvisAI:**
1. **Local-first** (Ollama offline) vs cloud-only competitors
2. **Voice-first PC automation** vs solo transcripción
3. **Flat pricing** (no per-minute charges) vs usage-based
4. **On-prem deploy** para Enterprise vs cloud lock-in

**Desventaja a mitigar:**
- Requiere Windows 11 + hardware decente (vs browser-based)
- Menos features cloud (no CRM nativo como Fireflies)

### Pricing Tiers Propuestos

#### **STARTER** — $79/mes (MXN $1,580/mes)
**Target:** Solopreneurs, freelancers, small teams (1-2 personas)

**Features core:**
- 1 usuario
- Voice control básico (open apps, web search, basic automation)
- Ollama local + Gemini API free tier
- Obsidian integration (read-only)
- 500 voice commands/mes
- Email support
- Updates incluidos

**Límite claro vs Pro:** Solo 1 usuario, no multi-agent orchestration, no custom skills

**Mensajing:** "Menos que un asistente virtual part-time ($200/sem)"

---

#### **PRO** — $199/mes (MXN $3,980/mes) ⭐ MOST POPULAR
**Target:** PyMEs establecidas (3-10 empleados), fundadores YC, consultoras

**Features core:**
- Hasta 5 usuarios
- Todo de Starter +
- Claude API integration (proxy or direct)
- Multi-agent orchestration (swarms)
- Custom skills builder
- Obsidian vault full access (read+write)
- CRM automation (Google Sheets, Notion, Airtable)
- Unlimited voice commands
- Priority support (24h response)
- Monthly usage analytics

**Límite claro vs Enterprise:** Max 5 usuarios, no on-prem deploy custom, no SLA 99.9%

**Mensajing:** "Precio de 1 empleado medio tiempo, productividad de 3 full-time"

---

#### **ENTERPRISE** — $799/mes (MXN $15,980/mes)
**Target:** Corporativos (10+ empleados), agencias, consultoras grandes

**Features core:**
- Usuarios ilimitados (flat pricing)
- Todo de Pro +
- On-premise deploy (corporate networks)
- SSO / Active Directory integration
- Custom LLM endpoint (BYO Claude/GPT keys)
- White-label branding
- Dedicated CSM (Customer Success Manager)
- SLA 99.5% uptime
- Custom integrations (Salesforce, SAP, legacy systems)
- Quarterly business review (QBR)
- Training sessions para equipo (2 horas/trimestre)

**Mensajing:** "Automatiza 40-60% overhead administrativo. ROI en 60 días."
