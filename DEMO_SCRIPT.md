## Notas de Producción

### Equipo necesario:
- ✅ Micrófono decente (no usar micrófono de laptop)
- ✅ OBS Studio o similar para screen recording
- ✅ Pantalla limpia (cerrar tabs/apps irrelevantes)

### Configuración OBS:
1. **Escena**: Captura de pantalla completa
2. **Audio**: Micrófono de Emmanuel (narración) + Audio de sistema (voz de Jarvis)
3. **Resolución**: 1920x1080 @ 30fps
4. **Bitrate**: 5000 kbps (calidad HD para YouTube)

### Tips de grabación:
- ✅ Ensayar el script 2-3 veces antes de grabar
- ✅ Hablar LENTO y claro (audiencia LATAM, incluye no-técnicos)
- ✅ Pausas de 2-3 seg entre secciones (facilita edición)
- ✅ Si te equivocas, NO parar grabación — simplemente reiniciar la frase (cortarás en edición)

### Post-producción (opcional):
- Cortar silencios largos con DaVinci Resolve o Adobe Premiere
- Agregar subtítulos en español (YouTube auto-generate + corrección manual)
- Background music sutil (royalty-free: Epidemic Sound, Artlist)
- Overlay de texto para highlights ("🔥 20 minutos → 10 segundos")

## Alternativa: Demo Automatizado

Si prefieres no narrar en vivo, usa `demo_recorder.py`:

```bash
python demo_recorder.py
```

Esto graba automáticamente las 3 tareas sin necesidad de voz over. Luego:
1. Agregas narración con herramienta de video (iMovie, DaVinci)
2. O dejas solo música de fondo + texto overlay

**Ventaja:** Más rápido (no necesitas ensayar)  
**Desventaja:** Menos personal (la voz de Emmanuel vende más que texto)
