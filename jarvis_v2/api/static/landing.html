Preguntas honestas

¿Es seguro que una IA controle mi computadora?

Sí. Cada acción se verifica visualmente antes y después con SSIM (Structural Similarity Index). Si el sistema no detecta el cambio esperado, NO continúa. Tiene un tribunal interno (Actor-Critic) que audita código de alto riesgo antes de ejecutarlo. Y nunca toca tu mouse sin tu permiso explícito.

¿Mis datos se mezclan con los de otros clientes?

No. Cada cliente tiene su propio archivo de base de datos aislado en disco (data/tenants/{tu_id}/). Las API keys se guardan cifradas con SOPS+age fuera de la base de datos. Cumple el patrón multi-tenant que pide SOC 2.

¿Qué pasa si la IA se equivoca?

El sistema registra cada acción en SQLite con éxito/error y métrica SSIM. Tienes un dashboard donde ves exactamente qué hizo, qué falló, y por qué. La memoria persistente aprende de los errores para no repetirlos.

¿Cuánto pago de extra a Anthropic / OpenAI por las llamadas IA?

Cero extra. El plan que pagas incluye todas las llamadas IA. Usamos OAuth Max con suscripción flat, sin pay-per-token al cliente final.
