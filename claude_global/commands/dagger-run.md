Ejecuta pipelines CI/CD con Dagger.

Comandos:
- Listar funciones: `dagger functions`
- Ejecutar: `dagger call FUNCION`
- Inicializar modulo: `dagger init`

Dagger es CI/CD programable - define pipelines en codigo (Go, Python, TypeScript) en vez de YAML. Requiere Docker running.

Si no hay dagger.json, ofrecer inicializar el proyecto con `dagger init`.
